# Pokete 操作指南

## 游戏目标

你扮演一名训练师，在地图中探索、遭遇野生 Pokete、捕捉它们、培养队伍，并挑战其他训练师。

## 基础操作

### 地图移动

- `W` / `A` / `S` / `D`：上下左右移动
- 角色在地图上显示为一个小写的 `a`

### 通用菜单操作

- `W` / `S`：上下移动光标
- `Enter`：确认
- `Esc` 或 `q`：退出当前界面

### 主菜单

- `e`：打开菜单/设置

菜单里通常可以进行这些操作：

- 修改玩家名
- 保存游戏
- 退出游戏
- 调整其他设置

注意：README 提到，对队伍等内容的改动通常需要通过正常退出功能保存。

## 开局怎么玩

1. 启动后先创建角色并获得初始 Pokete。
2. 在地图上用 `WASD` 探索。
3. 进入草地 `;` 时，可能遇到野生 Pokete。
4. 通过战斗降低对方血量后，用 Poketeball 捕捉。
5. 在城镇和路线间移动，挑战训练师、进入中心回血、去商店补给。

## 战斗操作

进入战斗后，常见流程如下。

### 攻击

- `1`：打开攻击列表
- 然后用 `W` / `S` 选择攻击，按 `Enter` 使用
- 或直接按攻击前面的数字快速选择

每个技能后面的数字是 AP。

- AP 表示这个技能还能使用多少次
- AP 用完后，这个技能暂时不能再用

### 捕捉野生 Pokete

先把对手血量压低，再尝试捕捉，成功率会更高。

- 在战斗里按 `3` 打开物品栏
- 选择 Poketeball
- 按 `Enter` 使用

如果捕捉失败，可以继续削弱对方后再次尝试。

### 更换 Pokete 或逃跑

战斗界面会提供攻击、使用物品、换怪、逃跑等选项。

- 野外战斗通常可以逃跑
- 训练师战斗不能逃跑

### 升级与学习技能

- 打倒敌人后会获得经验值
- 每隔若干次升级，Pokete 可能学会新技能
- 如果技能已满 4 个，系统会让你选择替换旧技能

## 地图中的重要内容

### 草地

- 地图中的草地用 `;` 表示
- 走进草地可能触发野生战斗

### 训练师

- 其他训练师在地图上也可能显示为 `a`
- 靠近后会对话并发起战斗
- 训练师战斗不能逃跑，也不能捕捉对方的 Pokete
- 胜利后通常能获得奖励和更多经验

### Poketeball

- 地图上的红球就是 Poketeball
- 直接走过去就能拾取
- 它们是捕捉野生 Pokete 的必要道具

## 各功能界面

### 队伍/Deck

README 中提到：

- `1` 也可以在主地图中查看当前队伍
- 可以查看 Pokete 详情
- 可以调整队伍顺序

这部分和战斗里的 `1` 都与当前可用操作相关，看到界面提示后按说明选择即可。

### 背包

- `4`：打开背包
- 可以查看道具和金钱数量
- 选中道具后按 `Enter` 查看说明
- 按 `r` 可移除当前选中的道具
- `Esc` 或 `q` 退出
y
### 小地图

- `3`：打开世界地图/小地图
- 用 `W` / `A` / `S` / `D` 在地图中移动选择
- `Esc` 或 `q` 退出

注意：`3` 在不同场景下作用不同。

- 主地图中：打开世界地图
- 战斗中：打开物品栏

### Poketedex

- `5`：打开 Poketedex
- 用来查看你捕获过的 Pokete 图鉴信息
- 没抓到过的种类通常会显示为 `???`

## 建筑说明

### Pokete Center

Pokete Center 类似回复点。

- 进入建筑
- 走到柜台人物前进行互动
- 可以治疗受伤或倒下的 Pokete
- 也可以整理你的队伍

### Shop

商店里可以购买：

- Poketeball
- Potion 等消耗品

操作方式：

- 走到柜台人物前互动
- 用方向键选择商品
- 按 `Enter` 购买
- `Esc` 或 `q` 退出

## 联机交易

如果你和另一名玩家在同一网络内，可以交易 Pokete。

大致流程：

1. 在 Pokete Center 找到交易 NPC。
2. 一名玩家作为 host，另一名作为 guest。
3. host 先发起交易。
4. guest 输入 host 屏幕上显示的主机名。

如果失败，常见原因可能是网络或防火墙限制。

## 稀有内容

### Shiny Pokete

- Shiny 很稀有
- 属性会比普通版本更强一些
- 名字通常会以特殊颜色高亮显示

### Learning Disc

- 这是一种一次性道具
- 可用于教会 Pokete 特殊攻击

## 新手建议

1. 先熟悉 `WASD` 移动和 `e` 菜单。
2. 遇到野怪后先学会用 `1` 攻击。
3. 血量压低后再用 Poketeball 抓，别一上来就扔。
4. 多捡地图上的红球，避免 Poketeball 不够用。
5. 受伤后及时去 Pokete Center 回复。
6. 不要全屏运行，项目文档明确说全屏下可能显示异常。

## 一页速查

- `W` / `A` / `S` / `D`：移动
- `e`：菜单/设置
- `1`：战斗中打开攻击列表；主地图中可查看队伍
- `3`：主地图中打开世界地图；战斗中打开物品栏
- `4`：打开背包
- `5`：打开 Poketedex
- `Enter`：确认
- `Esc` / `q`：返回或退出当前界面
- `r`：在背包中移除选中道具
