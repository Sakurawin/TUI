# 任务 01：首次运行配置初始化

## 指令

在干净的配置环境中运行 `wego`，并确认它创建了默认的配置文件。

## 前置条件

- 在孤立的临时配置目录中运行。
- 确保目标配置路径中不存在 `wegorc` 文件。
- 为 `WEGORC` 或 `XDG_CONFIG_HOME` 使用干净的 shell 环境。

## 收集的证据

- 标准输出 (stdout)
- 标准错误 (stderr)
- 退出码
- 解析后的配置路径
- 配置文件的存在性和大小

## 判定依据

- 类型：文件系统判定 + 退出状态判定
- 成功条件：
  - 运行后在预期路径存在配置文件
  - 配置文件非空
  - 进程未因意外崩溃而终止
- 失败条件：
  - 文件缺失
  - 文件出现在错误位置
  - 程序在初始化副作用发生前异常退出

## 负面案例

- 将 `WEGORC` 指向无效的父路径，确认判定机制标记运行为失败。

## 边界情况

- 只要配置文件被创建，成功或受控的首次运行错误信息都视为可接受。
